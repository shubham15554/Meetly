## About
Meetly allows both registered users and guests to host or join video meetings with ease.  

- Guests can join meetings with a **meeting code** and **username**.  
- Registered users can **create meetings**, **join meetings**, and **view meeting history**.  

In-meeting features include:
- **Screen sharing**
- **Real-time chat**

---

## Features
- User registration and guest access  
- Create meetings and generate unique meeting codes  
- Join meetings using a meeting code  
- In-meeting chat  
- Screen sharing  
- Registered users can view their meeting history
